# Texto de Apresentação — Orbital Memory

Olá, meu nome é Ana Carolina Ramalho de Matos, sou estudante de Engenharia de Software da FIAP, e este é o projeto **Orbital Memory**, desenvolvido para a Global Solution 2026, com o tema **“Indústria Espacial: O Código que Move o Universo”**.

O Orbital Memory surgiu da seguinte questão: se aviões possuem caixas-pretas para preservar informações após acidentes, por que cidades ainda podem perder dados críticos durante desastres?

Em situações como enchentes, incêndios, deslizamentos e falhas de comunicação, informações essenciais podem ser perdidas justamente quando são mais necessárias. O Orbital Memory foi criado para atuar como uma **caixa-preta urbana**, registrando, organizando e preservando dados importantes antes, durante e após eventos extremos.

Inspirado em sistemas espaciais, o projeto utiliza conceitos de telemetria, monitoramento remoto e redundância de dados. Assim como missões espaciais precisam continuar operando em condições adversas, o Orbital Memory busca manter informações disponíveis mesmo em momentos de crise.

Na etapa de **Software e Total Experience Design**, foi definido o escopo da solução, incluindo problema, público-alvo, jornada do usuário e backlog de requisitos.

Na parte de **Front-End Design e Web Development**, foi desenvolvido um dashboard interativo com indicadores de eventos, sensores ativos, áreas monitoradas e alertas. A plataforma também conta com mapa de regiões, histórico de ocorrências e simulações de cenários como enchentes, incêndios e falhas de comunicação.

Em **Computational Thinking with Python**, foi criado um sistema capaz de analisar dados simulados de sensores urbanos, classificar riscos e gerar relatórios automáticos com base em informações como nível do rio, temperatura, umidade e status de comunicação.

Também foi realizada uma análise matemática da eficiência do sinal de satélite utilizando derivadas e o método de Newton-Raphson, auxiliando na definição de uma altura adequada para a antena da estação.

Além disso, o projeto inclui uma estação física simulada com Arduino, sensores, display LCD, LEDs e buzzer, capaz de identificar situações normais, de atenção e de risco de forma autônoma.

O Orbital Memory contribui para cidades mais resilientes e inteligentes, alinhando-se ao **ODS 11 — Cidades e Comunidades Sustentáveis**. Seu objetivo é garantir que informações essenciais permaneçam acessíveis para apoiar decisões rápidas e eficientes durante emergências.

**Orbital Memory: nenhum desastre deve apagar a memória de uma cidade.**
